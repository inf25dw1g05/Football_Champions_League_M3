# C1 : Champions League Manager - Desenvolvimento Web 1
 
## Description
 
No âmbito da Unidade Curricular de Desenvolvimento Web I, este projeto consiste no desenvolvimento de uma solução "Full Stack" para a gestão de uma competição desportiva, especificamente uma fase de grupos da Liga dos Campeões.
 
O objetivo principal foi a implementação de uma arquitetura robusta baseada em serviços RESTful, permitindo a utilização completa dos métodos CRUD (Create, Read, Update, Delete) sobre as entidades principais: Equipas, Jogadores, Grupos, Jogos e Eventos de Jogo (golos, sanções disciplinares e assistências).
 
A solução divide-se em três componentes fundamentais:
1.  **Backend (API)**: Desenvolvido em **LoopBack 4**, seguindo o padrão *Controller-Repository-Model*. Responsável pela lógica de negócio e exposição dos dados.
2.  **Frontend (Backoffice)**: Desenvolvido em **React Admin**, consumindo a API para oferecer uma interface gráfica intuitiva para a gestão dos dados.
3.  **Persistência de Dados**: Base de dados relacional **MySQL**, configurada com chaves estrangeiras e "Triggers" para automatização de cálculos (como a atualização da tabela de classificação em tempo real).
 
Para garantir a portabilidade e facilidade de execução em qualquer ambiente, o projeto utiliza uma arquitetura **multicontainer Docker**, orquestrando os serviços da API, do Backoffice e da Base de Dados num único comando.
